## == && ===
== 和 === 的区别在于 == 允许在类型不相等的情况下 进行类型转换 而 === 不予许转换 类型不同直接返回false

== 的规则
```
1.  null == undefined 不==任何其他值 记住这是一对
2. NaN Symbol 不和任何值相等 包括自己
3. 复杂类型和基本类型比较 把复杂类型先转换为基本类型
4. 基本类型之间如果类型不同 都转换为数字类型比较
5. 基本类型之间如果类型相同 直接比较
6. 两边都是复杂类型 比较地址
```
