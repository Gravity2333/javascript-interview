
### 转换为 Symbol

Symbol 转换成别的类型 还是别的类型转换为 Symbol 都必须显转
Symbol 转换为其他类型 number❌ boolean true 字符串 "symbol(desc)"
其他类型转换为 Symbol <=> 调用 Symbol 的构造函数

```Javascript
function Symbol(desc){
    if(desc === undefined) return makeSymbol(undefined) // undefined不会转换为symbol('undefined') 但是null会 symbol('null')

    if(typeof desc === 'symbol') throw new Err

    desc = @@toPrimitive(desc,'string')

    makeSymbol(desc)
}
```
