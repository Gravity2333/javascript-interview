## 常见状态码

### 1xx

表示请求过程还没结束 还有下一步的操作 比如
- HTTP 100 大文件上传 用来在上传文件之前 鉴权
- 101 升级协议 常见 http -> websocket
  connection: upgrade
  upgrade: websocket

### 2xx

表示请求成功

- 200 请求成功
- 201 表示服务器成功创建资源
- 204 No Content 表示请求成功，但是没有返回任何内容
- 206 Partial Content 部分返回内容 常用于断点续传
  Accepted-Range: byte
  Content-Range: bytes=start-end/total

  Range: bytes=start-end


# 3xx
- 301 表示永久迁移 需要结合location重定向 浏览器会缓存 下次直接访问location
- 302 表示临时迁移 需要结合location重定向
- 304 使用缓存 一般用于协商缓存
- 307 功能和302一样 但是禁止重定向过程中 post->get
302 由于没有标准的约束 所以浏览器在比如提交表单的时候 为了避免再次提交 会把post->get
而 307 强制约束 不能把post->get

# 4xx
- 400 Bad Request 请求格式错误 URL包含非法字符 content-type和内容不符等
- 401 UnAuthorized 未授权 常见于用户未登录 或者登陆过期 登陆失败
- 403 用户无权限 常用于用户登录了 但是没有权限操作 读取 某些资源 比如普通用户没有权利创建用户
- 404 资源未找到 Not Found 很常见了
- 416 Range UnSatified 分片下载不被满足 常见于断点续传

# 5xx
- 500 服务器错误
- 502 网关错误 有可能是网关自己错误 也有可能是上游服务器返回的有问题
- 504 网关超时 有可能是网关自己超时 也可能是上游服务器返回超时
